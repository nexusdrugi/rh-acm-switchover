# NAME
acm-switchover-state - View and explain switchover state files

# SYNOPSIS
**acm-switchover-state** [--list] [--json] [STATE_FILE]

# DESCRIPTION
Lists or displays switchover state files with human-friendly output and JSON mode.

# OPTIONS
--list, --json, --no-color, --version

# ENVIRONMENT
ACM_SWITCHOVER_STATE_DIR - default state directory when STATE_FILE is not provided.

# SEE ALSO
acm-switchover(1), acm-switchover-rbac(1)
